# E-Commerce Platform

This is a comprehensive e-commerce solution designed to provide a seamless shopping experience.

## Features

- **User Authentication**: Secure login and registration using NextAuth.js.
- **Product Catalog**: Dynamic product listing with filtering and sorting.
- **Shopping Cart**: Real-time cart management.
- **Checkout**: Integrated Stripe payment gateway.
- **Admin Dashboard**: Manage products, orders, and customers.

## Tech Stack

- **Frontend**: Next.js, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL with Prisma
- **Payments**: Stripe
