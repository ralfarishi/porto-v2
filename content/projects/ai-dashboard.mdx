# AI Analytics Dashboard

This project provides a real-time overview of AI model metrics. It integrates with TensorFlow serving to fetch inference statistics and visualizes them using Recharts.

## Features

- Real-time latency monitoring
- Accuracy tracking
- Resource usage graphs
- Alerting system

:::info
The dashboard updates every 5 seconds to ensure you have the latest data.
:::

To install the dependencies, run:

```bash
npm install
```

Make sure to configure your `API_KEY` in the `.env` file.
