# Mastering Next.js 14

Next.js 14 introduces several groundbreaking features that change how we build web applications.

## Server Actions

Server Actions allow you to execute server-side code directly from your components without creating separate API routes.

:::info
Colors like `text-fd-muted-foreground` are from the design system of Fumadocs UI, you may also use your own colors, or use Shadcn UI.
:::

```tsx filename="next.config.mjs"
import createMDX from "fumadocs-mdx/config";

const withMDX = createMDX();

/** @type {import('next').NextConfig} */
const config = {
	reactStrictMode: true,
};

export default withMDX(config);
```
