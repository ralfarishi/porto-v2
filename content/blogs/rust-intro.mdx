# Introduction to Rust for JS Developers

Rust has been voted the most loved programming language for several years in a row. But is it worth learning for a JavaScript developer?

## Memory Safety

One of Rust's main selling points is its memory safety guarantees without a garbage collector.

:::info
Rust's ownership model is unique and may take some time to get used to.
:::

Here's a simple "Hello, World!" in Rust:

```rust
fn main() {
    println!("Hello, world!");
}
```

## Cargo

Cargo is Rust's package manager, similar to `npm` in the JavaScript ecosystem.
