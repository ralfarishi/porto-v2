# Optimizing React Performance

Performance is key to a good user experience. In this guide, we'll cover common pitfalls and how to avoid them in your React apps.

## Memoization

Using `useMemo` and `useCallback` effectively can prevent unnecessary re-renders.

:::danger
Don't optimize prematurely! Measure performance issues before applying fixes.
:::

Example of `useMemo`:

```javascript
const expensiveValue = useMemo(() => {
	return computeExpensiveValue(a, b);
}, [a, b]);
```

## Virtualization

For long lists, consider using windowing libraries like `react-window`.
